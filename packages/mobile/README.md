## üöÄ Getting started

1.  **Clone the repository**

    ```sh
    git clone https://github.com/GSTJ/XPerion
    cd XPerion/mobile
    ```

2.  **Install the necessary packages.**

    ```sh
    yarn
    ```

3.  **Start developing.**

    Initiate the project in development mode

    ```js
    // Android
    yarn run android
    
    // IOS
    yarn run ios
    ```
